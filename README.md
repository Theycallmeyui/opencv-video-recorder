# opencv-video-recorder
OpenCV와 Python을 이용하여 웹캠 영상을 녹화하는 간단한 프로그램이다.

# 프로그램 설명
이 프로그램은 컴퓨터의 웹캠을 이용하여 실시간 영상을 보여주고,
사용자가 키보드를 이용하여 녹화를 시작하거나 종료할 수 있다.

또한 추가 기능으로 흑백(Grayscale) 필터를 적용할 수 있다.

# 주요 기능
- 웹캠 실시간 영상 표시
- Space 키로 녹화 시작 / 종료
- 녹화 중 화면에 REC 표시
- ESC 키로 프로그램 종료
- G 키로 흑백 필터 ON / OFF

# 사용 방법
1. Python과 OpenCV를 설치한다.
2. 프로그램을 실행한다.


# 키보드 조작
|   키  |   기능  |
|-------|-------|
| Space | 녹화 시작 / 종료 |
|   G   | 흑백 필터 ON / OFF |
|  ESC  | 프로그램 종료 |
